## 6. FUNIL DE CONVERSÃO — LÓGICA HORMOZI

Baseado na trilogia: **$100M Offers · $100M Leads · $100M Money Models**

### Princípios aplicados ao projeto

**Grand Slam Offer (100M Offers):**
- A oferta não pode ser comparada — artesanato em vidro com identidade visual única é difícil de replicar
- O produto digital deve ter percepção de valor alta + baixo atrito de entrada
- Garantia clara e remoção de risco para converter a audiência fria

**Leads (100M Leads):**
- O tráfego orgânico já existe (14M views) — o problema é captura, não geração de atenção
- Lead magnet = porta de entrada para a lista (e-mail / WhatsApp)
- Automação de DM por palavra-chave = cada comentário vira uma oportunidade de captura

**Money Models (100M Money Models):**
- Não depender de uma única fonte de receita
- Construir a lista como ativo permanente de Augusto (não da plataforma)
- Evergreen > lançamento pontual — receita previsível ao longo do tempo

### Funil principal (evergreen)
```
Reels viral (topo) 
  → CTA para link na bio 
    → Página de captura (e-mail / WhatsApp)
      → Sequência de aquecimento (3–5 e-mails / mensagens)
        → Oferta low-ticket (R$27–97)
          → Upsell curso (R$297–597)
            → Upsell mentoria (High-ticket)
```

### Lançamento Meteórico (injeção rápida de caixa)
- Canal: WhatsApp (grupos administrados pela Futurah)
- Janela: 5–7 dias de intensidade
- Sem webinar longo, sem sequência de vídeos — direto ao ponto
- Funciona para: curso, mentoria, drop físico

### Funil High-Ticket (perfil 2 — especialista)
- Atração seletiva: conteúdo que polariza e filtra curiosos
- Prova incontestável: métricas reais, bastidores do processo
- CTA: "Me manda MENTORIA no Direct" → qualificação pela Futurah
- Workshops tripwire (R$40–200) como pré-qualificação financeira
- Fechamento: call 1:1 com time comercial
